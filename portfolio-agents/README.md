# ⚡ Supreme Commander Portfolio Agency

> *"Commander, the enemy has made their first move. It is time to respond in kind."*

A superstar AI agent team assembled to build **David Lin's Three.js / Bun / SvelteKit portfolio website**, themed after the legendary RTS **Supreme Commander** by Chris Taylor and Gas Powered Games.

Each agent is a specialist drawn from one of the four factions of the Infinite War — engineered for a single purpose: constructing the ultimate portfolio on the digital battlefield.

## ⚡ Quick Deploy

### With GitHub Copilot / VS Code

The agents are `.md` files with YAML frontmatter. Reference them directly in your Copilot chat:

```
Use the Supreme Commander agent from portfolio-agents/supreme-commander.md to plan the initial architecture.
```

### With Claude Code

```bash
cp -r portfolio-agents/* ~/.claude/agents/
```

### With Cursor / Windsurf

Copy individual agent files into your `.cursorrules` or agent configuration, or reference them as context files in your session.
